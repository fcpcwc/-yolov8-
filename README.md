# 行人车辆密集检测系统

基于 YOLOv8 深度学习模型开发的智能目标检测系统，专注于复杂场景下的行人与车辆识别与计数。

## 功能特性

- **多模式检测**：支持图片、视频文件及摄像头实时检测
- **目标跟踪**：基于 BoT-SORT 算法实现多目标轨迹跟踪
- **密集度热力图**：可视化人群/车辆聚集密度分布
- **参数可调**：支持置信度阈值、IOU阈值自定义设置
- **模型灵活**：支持切换不同 YOLO 模型文件
- **结果保存**：自动保存检测结果图片/视频

## 支持的检测类别

| 类别 | 英文名    | 中文名 |
| ---- | --------- | ------ |
| 0    | aeroplane | 飞机   |
| 1    | bicycle   | 自行车 |
| 2    | bus       | 公交车 |
| 3    | car       | 小汽车 |
| 4    | motorbike | 摩托车 |
| 5    | person    | 行人   |
| 6    | train     | 火车   |

## 技术栈

- **模型框架**：Ultralytics YOLOv8
- **目标跟踪**：BoT-SORT（回退至 scipy 线性分配）
- **界面开发**：PyQt5
- **图像处理**：OpenCV、NumPy

## 快速开始

### 环境要求

- Python 3.8+
- CUDA（可选，GPU 加速）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python MainProgram.py
```

## 项目结构

```
├── MainProgram.py          # 主程序入口
├── Config.py               # 配置文件
├── detect_tools.py        # 检测工具函数
├── requirements.txt       # 依赖列表
├── UIProgram/             # 界面相关代码
│   ├── UiMain.py         # 主界面逻辑
│   ├── style.css         # 界面样式
│   └── ui_imgs/          # 界面图片资源
├── models/                # 模型文件目录
├── ultralytics/          # YOLO 框架源码
└── save_data/            # 检测结果保存目录
```

## 使用说明

1. **图片检测**：点击"图片检测"按钮选择本地图片文件
2. **视频检测**：点击"视频检测"按钮选择视频文件
3. **摄像头检测**：点击"摄像头"按钮开启实时检测
4. **目标导航**：检测到多个目标时，使用左右按钮切换浏览
5. **切换模型**：点击"选择模型"更换不同的 YOLO 模型
